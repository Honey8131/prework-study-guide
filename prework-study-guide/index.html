Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents visible content show to user
  • There are opening and closing tags

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Always make sure you link the HTML file to the CSS file

Git

  • Git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -a: add new feature branch to current working branch
  • got commit -m "message": write description of what you did
  • git pull origin file-name: pull changes from repo to make sure you're in sync
  • git push origin file-location: push local changes to remote branch
